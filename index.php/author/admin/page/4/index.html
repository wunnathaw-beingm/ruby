正規表現を使うメソッド

Stringクラスには正規表現と組み合わせと便利に使えるメソッドが用意されています。

■scan
scanメソッドは引数で渡した正規表現にマッチする部分を配列に入れて返します。

'123 456 789'.scan(/\d+/)  #=> ["123", "456", "789"]

■[], slice, slice!
[]に正規表現を渡すと、文字列から正規表現にマッチした部分を抜き出します。

text = 'post-no 123-4567'
text[/\d{3}-\d{4}/]   #=> "123-4567"

#マッチする部分が複数ある場合、最初にマッチした文字列が返ります。
text = '123-4567 456-7890'
text[/\d{3}-\d{4}/]   #=> "123-4567"

sliceメソッドは[]のエイリアスメソッドです。

text = 'post-no 123-4567'
text.slice(/\d{3}-\d{4}/)   #=> "123-4567"

#マッチする部分が複数ある場合、最初にマッチした文字列が返ります。
text = '123-4567 456-7890'
text.slice(/\d{3}-\d{4}/)  #=> "123-4567"

slice!にするとマッチした部分が文字列から破壊的に取り除かれます。

text = 'post-no 123-4567'
text.slice!(/\d{3}-\d{4}/)   #=> "123-4567"
text     					 #=> "post-no"

■split
splitに正規表現を渡すと、マッチした文字列を区切り文字にして文字列を分解し、配列として返します。

text = '123,456-789'
text.split(/,|-/)   #=> ["123", "456", "789"]

■gsub、gsub!
第1引数の正規表現にマッチした文字列を第2引数の文字列で置き換えます。

text = '123,456-789'
text.gsub(/,|-/, ':')    #=> "123:456:789"

#第2引数にハッシュを渡して、変換のルールを指定することもできます。
hash = {',' => ':', '-' => '/'}
text.gsub(/,|-/, hash)   #=> "123:456/789"

gsub!メソッドは文字列の内容を破壊的に置き換します。

text = '123,456-789'
text.gsub!(/,|-/, ':')    #=> "123:456:789"
text                      #=> "123:456:789"

正規表現オブジェクトの生成

Rubyにおける正規表現オブジェクトは次のようにスラッシュでパターンを囲んで作成します。

regex = /\d{3}-\d{4}/
regex.class     #=> Regexp

Regexp.new()を使って正規表現オブジェクトを作成できます。

Regexp.new("Ruby")     #=> /Ruby/

もう一つの方法は%rを使って作成することです。

%r(/Ruby/)     #=>  /\/Ruby\//

正規表現と文字列を比較する場合は =~ を使って比較できます。マッチした場合は文字列中のマッチした位置が返り、マッチしなかった場合はnilが返ります。

#マッチした場合はマッチした文字列の開始位置が返る
'123-456' =~ /\d{3}-\d{4}/   #=> 0

#マッチしない場合nilが返る
'hello' =~ /\d{3}-\d{4}/    #=> nil

シンボルの特徴

シンボルをハッシュのキーとしてよく使われています。ハッシュのキーにシンボルを使うと、文字列よりも高速に値を取り出すことができます。
以下は文字列をハッシュのキーにした例です。

capital = { 'japan' => 'tokyo', 'myanmar' => 'naypyitaw' }

シンボルをハッシュのキーにすると以下のようになります。

capital = { :japan => 'tokyo', :myanmar => 'naypyitaw' }
#シンボルを使って値を取り出す
capital[:japan]   #=>  "tokyo"

シンボルがキーになる場合、=> を使わずに”シンボル：値”と言う記法でハッシュを作成できます。

capital = { japan: 'tokyo', myanmar: 'naypyitaw' }
#シンボルを使って値を取り出す
capital[:japan]   #=>  "tokyo"

キーも値もシンボルの場合は次のようになります。

capital = { :japan => :tokyo, :myanmar => :naypyitaw }
#シンボルを使って値を取り出す
capital[:japan]   #=>  :tokyo

=>を使わない場合、コロン(:)同士が向き合う形でハッシュを作成できます。

capital = { japan: :tokyo, myanmar: :naypyitaw }
#シンボルを使って値を取り出す
capital[:japan]   #=>  :tokyo
投稿日: 作成者 カテゴリー Rubyに関する基礎知識, シンボル

シンボルの定義

シンボルの定義方法は(:)に続けて任意の名前を付けます。

symbol = :name
symbol      #=> :name

%記法を使ってシンボルを作成することができます。%sを使って以下のように作成できます。

#!を区切り文字に使う
%s!ruby is fun!    #=>  :"ruby is fun"

%s(ruby is fun)    #=>  :"ruby is fun"

%iを利用してシンボルの配列を作成できます。空白文字が要素の区切りになります。

%i(japan myanmar korea)    #=>   [:japan, :myanmar, :korea]

%iでは改行文字や式展開の構文が、そのままシンボルになります。

name = 'Alice'
%i(hello\ngood-bye #{name.upcase})   #=>  [:"hello\\ngood-bye", :"\#{name.upcase}"]

%Iを利用すると改行文字や展開が有効になり、シンボルが作成できます。

name = 'Alice'
%I(hello\ngood-bye #{name.upcase})   #=> [:"hello\ngood-bye", :ALICE]

ハッシュのさまざまな操作

■ハッシュを作成する方法

ハッシュとはキーと値の組み合わせでデータを管理するオブジェクトです。
以下はハッシュを作成する例です。

{
  'japan' => 'yen',
  'myanmar' => 'kyats',
  'us' => 'dollar'
}

同じキーが複数使われた場合、最後に出てきた値が有効になります。

{
    'japan' => 'yen',
    'japan' => '円'
}
#=> { "japan" =>  "円"}

■ハッシュのデータを取得する方法

currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies['japan']  #=> "yen"

■ハッシュに要素を追加

以下は新しい要素を追加する例です。
キーが存在しない場合は、新たに追加されます。

currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies['thai'] = 'baht'
currencies   #=> { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar', 'thai' => 'bhat' }

storeメソッドを利用して、指定したキーに対する新しい値を代入することができます。

# hash.store(key, value)
currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies.store('thai', 'baht')
currencies     #=> { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar', 'thai' => 'bhat' }

■ハッシュの要素の更新

ハッシュに存在するキーを指定値を代入すると値が上書きされ、更新されます。

currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies['japan'] = '円'
currencies     #=> { 'japan' => '円', 'myanmar' => 'kyats', 'us' => 'dollar' }

■ハッシュの要素の削除

deleteメソッドを利用すると指定したキーに対する要素を削除できます。

currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies.delete('us')
currencies      #=>  { 'japan' => 'yen', 'myanmar' => 'kyats' }

■ハッシュの繰り返し処理

eachメソッドを使って、キーと組み合わせを順に取り出すことができます。
ブロック引数がキーと値で2個の組み合わせになる必要があります。

hash = { 'apple' => 300, 'orange' => 200 }
hash.each { |key, value|
    puts "key: #{key}, value: #{value}"
}

#=>   key: apple, value: 300
#     key: orange, value: 200

ブロック引数を1つにするとキーと値が配列に格納されます。

hash = { 'apple' => 300, 'orange' => 200 }
hash.each { |fruits|
    key = fruits[0]
    value = fruits[1]
    puts "key: #{key}, value: #{value}"
}

#=>     key: apple, value: 300
#       key: orange, value: 200

■ハッシュと配列の交換

ハッシュから配列にto_aメソッドを利用して変換することができます。to_aメソッドを使うとキーと値が１つの配列になり、さらにそれが複数並んだ配列になります。

hash = { 'apple' => 300, 'orange' => 200 }
hash.to_a   #=> [["apple", 300], ["orange", 200]]

範囲(Range)

■範囲オブジェクトの指定方法

範囲オブジェクトは次のように..また…を使って作成します。
具体的には次のように作成します。
1..5 # 1から5まで
‘a’..’e’ #aからeまで
1…5　 #1から5まで
‘a’…’e’ #aからeまで

■case文で使う範囲

def charge(age)
    case age
    #0歳から5歳
    when 0..5
        0
    #6歳から12歳
    when 6..12
        300
    #13歳から18歳
    when 13..18
        600
    #それ以外
    else
        1000
    end
end

■配列で使う範囲

配列に対して添え字の代わりに範囲オブジェクトを渡すと、指定した範囲の要素を取得することができます。

a = [1, 2, 3, 4, 5]
a[1..3]    #=> [2, 3, 4]

■繰り返し処理で使う範囲

範囲オブジェクトを配列に変換して繰り返し処理を行うことができます。

numbers = (1..4).to_a
sum = 0
numbers.each { |n| sum += n}
sum #=> 10

範囲オブジェクトに対して直接eachメソッドを利用して繰り返し処理を行うこともできます。

sum = 0
(1..4).each{ |n| sum += n}
sum #=> 10

stepメソッドを利用して値を増やす間隔を指定できます。

numbers = []
(1..10).step(2) { |n| number << n }
numbers   #=> [1, 3, 5, 7, 9]

繰り返し処理

■timesメソッド

n回処理を繰り返したいと言う場合は、integerクラスのtimesメソッドを利用すると便利です。

sum = 0
#処理を10回繰り返す。nには0から9が入る
10.times { |n|
  sum += n
  }
 puts sum    #=> 45

■uptoメソッド

nからmまで数値を１つずつ増やしながら処理したい場合は、uptoメソッドを利用すると便利です。

1.upto(5) { |n| puts "count: #{n}" }
#=>     count: 1
#=>     count: 2
#=>     count: 3
#=>     count: 4
#=>     count: 5

■downtoメソッド

nからmまで数値を減らしていきたい場合は、downtoメソッドを利用すると便利です。

5.downto(1) { |n| puts "count: #{n}" }
#=>     count: 5
#=>     count: 4
#=>     count: 3
#=>     count: 2
#=>     count: 1

■stepメソッド

nからmまで数値をx個ずつ増やしていきたい場合は、stepメソッドを利用します。

1.step(10, 2) { |n| puts "count: #{n}"}
#=>     count: 1
#=>     count: 3
#=>     count: 5
#=>     count: 7
#=>     count: 9

■while文

while文は指定した条件が真である間、処理を繰り返します。

n = 0
while n < 5
    puts n += 1
end
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

条件式の後ろにdoを入れると1行で書けます。

n = 0
while n < 5 do puts n += 1 end
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

1行で書くのであれば修飾子としてwhile文を後ろに置いた方がスッキリ書けます。

n = 0
puts n += 1 while n < 5
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

どんな条件でも最低1回は実行したい場合は、begin…endで囲んでからwhileを書きます。

n = 0
begin
    n = 1
end while false
puts n    #=> 1

■until文

until文はwhile文と反対で、条件が偽である間繰り返し処理を行います。

num = 10
until num == 15 do
    puts num
    num += 1
end
#=>     10
#=>     11
#=>     12
#=>     13
#=>     14

■for文

for num in 1..5 do
  p num
end

#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

■loopメソッド

loopメソッドを使うと無限ループを作ることができます。無限ループから脱出する場合はbreakを使います。

num =0
loop{
  p num
  num+=2
  if num >= 10
    break
  end
}
  
#=>     0
#=>     2
#=>     4
#=>     6
#=>     8

■next

繰り返し処理を途中で中断し、次の繰り返し処理に進める場合はnextを使います。

[1,2,3,4].each do |num|
  next if num == 1
  p num
end

#=>   2
#=>   3
#=>   4

■redo

繰り返し処理をやりなおしたい場合はredoを使います。

[0,1,2,3,4].each do |num|
  num += 1
  p num
  redo if num == 1
end

#=>   1
#=>   2
#=>   2
#=>   3
#=>   4
#=>   5

配列のさまざまな操作

■配列を作成する方法

Rubyにおいて配列を作成する場合は、括弧（[])を使用します。値が複数ある配列は値をカンマ区切りで記述します。

fruit = ['apple']
fruits = ['apple', 'banana', 'orange']

異なるデータ型を格納することもできます。次は数値と文字列を混在させた配列です。

mixing = ['apple', 1, 'banana', 2, 'orange', 3]

配列の中に配列を含めることもできます。

net = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

■配列から要素を取得する方法

配列の各要素を取得する場合は、[]と添え字を使います。最初の要素の添え字は0です。

fruits = ['apple', 'banana', 'orange']
#1つ目の要素を取得する
fruits[0]   #=> "apple"
#2つ目の要素を取得する
fruits[1]   #=> "banana"
#3つ目の要素を取得する
fruits[2]   #=> "orange"

添え字を2つ使うと、添え字の位置と取得する長さを指定することができます。

配列[位置, 取得する長さ]

a = [1, 2, 3, 4, 5]
a[1, 3]      #=> [2, 3, 4]

values_atメソッドを使うと、取得したい要素の添え字を複数指定できます。

a = [1, 2, 3, 4, 5]
a.values_at(0, 2, 4)   #=> [1, 3, 5]

Rubyでは添え字を負の値が使えます。-1は最後の要素、-2は最後から2番目の要素というように指定できます。

a = [1, 2, 3]

#最後の要素
a[-1]    #=> 3
a[-2]    #=> 2

配列にlastとfirstというメソッドがあります。lastは配列の最後の要素を取得できます。引数に0以上の数値を渡すと、最後のn個の要素を取得できます。firstはlastの反対で先頭の要素を取得するメソッドです。

a = [1, 2, 3]

#lastメソッド
a.last   #=> 3
a.last(2)  #=> [2, 3]

#firstメソッド
a.first  #=> 1
a.first(2)  #=> [1, 2]

■配列に要素を追加する方法

配列に要素を追加する場合、push()メソッドが使えます。

a = [1, 2, 3]
a.push(4)    #=> [1, 2, 3, 4]

また、添え字を指定して、要素を追加することもできます。

a = [1, 2, 3]
a[3] = 4	#=> [1, 2, 3, 4]

また、[<<]記号を使用して、配列の末尾に要素を追加できます。

a = [1, 2, 3]
a << 4     #=> [1, 2, 3, 4]

■ 配列の要素を削除する方法

配列の要素を削除する場合、delete()メソッドが使えます。

a = [1, 2, 3]
a.delete(1)     #=> [2, 3]

指定位置の要素を削除する場合、delete_at()メソッドを使えます。

a = [1, 2, 3]
a.delete_at(1)   #=> [1, 3]

#存在しない添え字を指定するとnilが返る
a = [1, 2, 3]
a.delete_at(5)     #=> nil

■ 配列とブロックの使い方

eachメソッドを使って繰り返し処理を行うと何番目の要素を処理しているのか、ブロック内で判別できません。each_with_indexメソッドを使うと添え字を判別できます。

countries = ['japan', 'myanmar', 'america']
countries.each_with_index { |country, index| puts "#{index}:  #{country}" }
#=> 0: japan
#   1: myanmar
#   2: america

mapメソッドで繰り返し処理をしつつ、添え字も同時に取得したいときは
mapメソッドとwith_indexメソッドを組み合わせて使います。

countries = ['japan', 'myanmar', 'america']
countries.map.with_index { |country, index| "#{index}:  #{country}" }
#=> ["0:  japan", "1:  myanmar", "2:  america"]

countries.delete_if.with_index{ |country, index| country.include?('y') && index.odd? }
#=> ["japan", "america"]

with_methodはEnumeratorクラスのインスタンスメソッドです。map, each, delete_ifメソッドなど繰り返し処理を行うとメソッドのブロックを省略して呼び出すと、Enumeratorオブジェクトを返します。

countries.map       #=> #<Enumerator: ["japan", "myanmar", "america"]:map>
countries.each      #=> #<Enumerator: ["japan", "myanmar", "america"]:each>
countries.delete_if #=> #<Enumerator: ["japan", "myanmar", "america"]:delete_if>

配列の配列に対して、eachメソッドなどで繰り返し処理を実行すると、配列がブロック引数に渡ってきます。

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each do |dimension| 
    length = dimension[0]
    width = dimension[1]
    puts "length: #{length}, width: #{width}"
end
#=>     length: 10, width: 20
#=>     length: 30, width: 40
#=>     length: 50, width: 60

しかし、ブロック引数の数を2個にすると、値を別々に受け取ることができます。

dimensions.each do |length, width|
    puts "length: #{length}, width: #{width}"
end
#=>     length: 10, width: 20
#=>     length: 30, width: 40
#=>     length: 50, width: 60

each_with_indexを使ってブロック引数を2個にすると、第1引数で縦と横の値を配列として取得できます。

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each_with_index do |dimension, i| 
    length = dimension[0]
    width = dimension[1]
    puts "length: #{length}, width: #{width}, index: #{i}"
end
#=>     length: 10, width: 20, index: 0
#=>     length: 30, width: 40, index: 1
#=>     length: 50, width: 60, index: 2

配列の要素を受け取るブロック引数を()で囲むと、配列の要素を別々の引数として受け取ることができます。

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each_with_index do |(length, width), i|
    puts "length: #{length}, width: #{width}, index: #{i}"
end
#=>     length: 10, width: 20, index: 0
#=>     length: 30, width: 40, index: 1
#=>     length: 50, width: 60, index: 2

ブロック引数を；で区切り、続けて変数を宣言すると、ブロック内でのみ有効な独立したローカル変数を宣言することができます。

sum = 0
numbers = [10, 20]
numbers.each do |num; sum|
    sum = 1
    sum += num
    puts "In Block: #{sum}"
end
puts "Outer Block: #{sum}"

#=>     In Block: 11
#=>     In Block: 21
#=>     Outer Block: 0

等値を判断するメソッド

Rubyでは等値を判断するための equal ?とeql?メソッドがあります。

■equal?

equal?メソッドは同じインスタンスかどうかを判断する場合に使います。

a = 'abc'
b = 'abc'
a.equal?(b)         #=> false

c = a
a.equal?(c)         #=> true

■eql?

eql?メソッドは文字列の内容が等しいか判断する場合に使います。ハッシュで二つのキーが等しいかどうかを判定するのに使われます。

#文字列内容が等しいか判定する
a = 'abc'
b = 'abc'
a.eql?(b)        #=> true

#ハッシュの二つのキーが等しいか判定する
h = { 1 => 'Integer', 1.0 => 'Float' }
h[1]    #=>  "Integer"
h[1.0]  #=>  "Float"

#異なるキーとして扱われる
1.eql?(1.0)  #=> false

メソッド名の疑問符「?」と感嘆符「！」

■疑問符「?」

?で終わるメソッドは真偽値を返すメソッドを表します。

#空文字であればtrue、そうでなければfalse
''.empty?   #=> true
'abcd'.empty?    #=> false

#オブジェクトがnilであればtrue, そうでなければfalse
@foo.nil?     #=> true
@foo = 'foo'
@foo.nil?     #=> false

#奇数
1.odd?    #=> true

#偶数
2.even?     #=> true

■感嘆符「!」

!で終わるメソッドは破壊的メソッドを表します。メソッドを呼び出したオブジェクト自身が影響を受けることです。

array = [3, 1, 5, 2, 4]

a = array.sort
puts a   #=> [1, 2, 3, 4, 5]
puts array   #=> [3, 1, 5, 2, 4]

b = array.sort!
puts b   #=> [1, 2, 3, 4, 5]
puts array  #=> [1, 2, 3, 4, 5]
