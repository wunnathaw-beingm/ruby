例外処理のelse節

例外処理が発生しなかった場合にelse節を書いておくとelse節が実行されます。

begin
    # 例外処理が発生する可能性がある処理
rescue
    # 例外処理が発生した場合の処理
else
    # 例外処理が発生しなかった場合の処理
ensure
    # 例外の有無にかかわらず実行する処理
end

else節を使わなくても、begin節に例外が発生しなかった場合の処理を書いおくと良いので、else節が使われるケースは多くありません。

begin
    puts 'Hello World'
rescue
    puts 'exceptionが発生しました'
else
    puts 'exceptionが発生しませんでした'
end
#=> "Hello World"
#   "exceptionが発生しませんでした"

以下はelse節を使わない例です。出力されるのは同上です。

begin
    puts 'Hello World'
    puts 'exceptionが発生しませんでした'
rescue
    puts 'exceptionが発生しました'
end
#=> "Hello World"
#   "exceptionが発生しませんでした
