Class Variable

Ruby မှာ အောက်ပါအတိုင်း variable အမျိုးအစားတွေရှိကြပါတယ်
1・Local variable
2・Instance variable
3・Class instance variable
4・Class variable
5・Global variable
6・Constant

■Local variable

local variable အကြောင်းကို အရင် chapter တွေမှာ ပြောထားခဲ့ပြီဖြစ်လို့ ဒီနေရာမှာ မပြောတော့ပါဘူး

■Instance variable

class အတွင်းထဲမှာ instance variable ကိုသုံးနိုင်ပါတယ်. instance variable ကို တူညီတဲ့ object, တူညီတဲ့ instance ကနေ access လုပ်ပြီးခေါ်သုံးနိုင်ပါတယ်. 

class User
    def initialize(name)
        #instance variable ထဲကို parameter value ထည့်လိုက်ပါမယ်
        @name = name
    end

    def hello
        #instance variable ထဲက data ကို ထုတ်ပြပါမယ်
        "Hello, I am #{@name}."
    end
end

user = User.new('Alice')
user.hello     #=> "Hello, I am Alice."

■Class instance variable

instance variable ဆိုတာ @ နဲ့စပါတယ်.class instance variable ဆိုတာကတော့ class အတွင်းထဲမှာ ရေးထားတဲ့ variable ကိုဆိုလိုပါတယ်.method ရဲ့အတွင်းမှာ ရေးထားတဲ့ variable မဟုတ်ပါဘူး.method ရဲ့အတွင်းမှာ ရေးထားတဲ့ variable ဆိုရင် instance variable ဖြစ်သွားပါမယ်.
အောက်ပါ ဥပမာကို ကြည့်ပါ 

class Product
    #class instance variable
    @name = 'Product'

    def self.name
        #class instance variable
        @name
    end

    def initialize(name)
        #instance variable
        @name = name
    end

    def name
        #instance variable
        @name
    end
end

product = Product.new('movie')
product.name        #=> movie
Product.name        #=> Product

class instance variable ဖြစ်တဲ့အတွက် instance ဆောက်စရာမလိုဘဲ  class  ကနေ  Product.name ဆိုပြီးတော့ တိုက်ရိုက်ခေါ်သုံးလို့ရပါတယ်.

■Class variable

class variable ဟာ class အတွင်းထဲမှာ, class ရဲ့ method အတွင်းထဲမှာ,  super  class,  subclass တွေကနေလည်း ခေါ်သုံးနိုင်ပါတယ်.တစ်နေရာကနေ update လုပ်လိုက်ရင်  အကုန်လုံးလိုက်ပြောင်းသွားမှာဖြစ်ပါတယ်. class variable ဟာ @@ နဲ့စပါတယ်.
အောက်ပါ ဥပမာကို ကြည့်ပါ 

class Product
    #class instance variable
    @@name = 'Product'

    def self.name
        #class instance variable
        @@name
    end

    def initialize(name)
        #instance variable
        @@name = name
    end

    def name
        #instance variable
        @@name
    end
end

class DVD < Product
    @@name = 'DVD'

    def self.name
        @@name
    end

    def upcase_name
        @@name.upcase
    end
end

#DVD class ကိုဆောက်လိုက်တဲ့အချိန်မှာ @@name က DVD အဖြစ်ပြောင်းသွားတယ်.
Product.name   #=> "DVD"
DVD.name       #=> "DVD"

product = Product.new('great movie')
product.name     #=> "great movie"

#Product.new လုပ်လိုက်တဲ့အချိန်မှာ @@name က great movie အဖြစ်ပြောင်းသွားတယ်.
Product.name   #=> great movie
DVD.name       #=> great movie

dvd = DVD.new('awesome movie')
dvd.name     #=> "awesome movie"

#DVD.new လုပ်လိုက်တဲ့အချိန်မှာ @@name က awesome movie အဖြစ်ပြောင်းသွားတယ်.
Product.name   #=> awesome movie
DVD.name       #=> awesome movie

■Global variable

Ruby မှာ global variable ဆိုပြီးတော့ variable တစ်မျိုးရှိပါသေးတယ်. global variable ဟာဆိုရင် $ နဲ့စပါတယ်.global variable ဟာဆိုရင် class အတွင်းတွေ အပြင်တွေတင်မဟုတ်ဘဲ program ကြီးတစ်ခုလုံးရဲ့ဘယ်နေရာကမဆို ခေါ်သုံးလို့ရပါတယ်. 

$program_name = 'Awesome program'

class Program
    def initialize(name)
        $program_name = name
    end

    def self.name
        $program_name
    end

    def name
        $program_name
    end
end

#$program_name မှာ initialize လုပ်ထားခဲ့တဲ့ဟာထွက်လာပါတယ်
Program.name

program = Program.new('Super Program')
program.name       #=> Super Program

#Program.new လုပ်လိုက်တဲ့အချိန်မှာ $program_name က Super Program အဖြစ်ပြောင်းသွားတယ်.
Program.name       #=> Super Program
$program_name      #=> Super Program

■Constant

class အတွင်းထဲမှာ constant ကို define လုပ်လို့ရပါတယ်. 

class Product
    DEFAULT_PRICE = 0

    attr_reader :name, :DEFAULT_PRICE

    def initialize(name, price = DEFAULT_PRICE)
        @name = name
        @price = price
    end
end

product = Product.new('A free movie')
product.price       #=> 0

constant name ဟာဆိုရင် အမြဲတမ်း capital letter နဲ့ရေးရပါမယ်. constant ကို instance method ထဲကကော, class method ထဲကကော ခေါ်သုံးလို့ရပါတယ်.

constant ကို class အပြင်ကနေလည်းအောက်ပါအတိုင်း ခေါ်သုံးလို့ရပါတယ်. 

class Product
    DEFAULT_PRICE = 0
end

Product::DEFAULT_PRICE      #=> 0

constant  ကို class အပြင်ကနေ ခေါ်မသုံးစေချင်ရင် private_constant နဲ့ setting ချလို့ရပါတယ်. 

class Product
    DEFAULT_PRICE = 0

    #constant ကို private လုပ်လိုက်တာ
    private_constant :DEFAULT_PRICE
end

#private လုပ်ထားလိုက်လို့ အပြင်ကနေ ခေါ်သုံးလို့မရတော့ဘူး
Product::DEFAULT_PRICE      #=> NameError: private constant Product::DEFAULT_PRICE referenced

Ruby ရဲ့ constant ရဲ့ထူးခြားချက်တစ်ခုက constant value ကို change လို့ရပါတယ်. အဲ့လို change လိုက်ရင် (already initialized constant)ဆိုတဲ့ warning တော့ တက်ပါလိမ့်မယ်. 

class Product
    DEFAULT_PRICE = 0

    #constant value ကို change လိုက်တာ
    DEFAULT_PRICE = 1000
end
#=> warning: already initialized constant Product::DEFAULT_PRICE
#warning တက်ပြီးတော့ constant value က change သွားပါတယ်.
Product::DEFAULT_PRICE  #=> 1000

class အပြင်ကနေလည်း constant value ကို change လို့ရပါတယ်. 

class Product
    DEFAULT_PRICE = 0
end

#class အပြင်ကနေ constant value ကို change လိုက်တာ
Product::DEFAULT_PRICE = 3000

#=> warning: already initialized constant Product::DEFAULT_PRICE
Product::DEFAULT_PRICE  #=> 3000

‌နောက်တစ်ခုအနေနဲ့ constant ကို assign မလုပ်သော်လည်း constant value ပြောင်းသွားစေတဲ့အရာတစ်ခုရှိပါတယ်.mutable object (String, Array, Hash) တို့ဖြစ်ပါတယ် 

class Product
    SOME_NAMES = ['Foo', 'bar', 'Baz']

    def self.names_without_foo(name = SOME_NAMES)
        #names ကိုသာ default value အနေနဲ့ထားရင် အောက်က code က SOME_NAMES ရဲ့ value ကိုပြောင်းသွားစေမှာပါ
        names.delete('Foo')
        names
    end
end

Product.names_without_foo   #=> ["Bar", "Baz"]

#constant value က ပြောင်းသွားပါပြီ
Product.SOME_NAMES          #=> ["Bar", "Baz"]
