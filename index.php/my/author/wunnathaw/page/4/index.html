Regular Expression Methods

String class မှာ Regular Expression အတွက်အသုံး၀င်တဲ့ method တွေ ရှိပါတယ်.

■scanscan method ဟာဆိုရင် parameter ပေးလိုက်တဲ့ Regular Expression နဲ့ match ဖြစ်တဲ့အပိုင်းကိုarray အနေနဲ့ ပြန်ပေးမှာ ဖြစ်ပါတယ်. 

'123 456 789'.scan(/\d+/)  #=> ["123", "456", "789"]

■[], slice, slice![]ထဲမှာ Regular Expression ကိုထည့်‌ပေးလိုက်ရင် string ထဲကနေRegular Expressionနဲ့match ဖြစ်တဲ့ဟာကို ထုတ်ပေးမှာဖြစ်ပါတယ်. 

text = 'post-no 123-4567'
text[/\d{3}-\d{4}/]   #=> "123-4567"

#match ဖြစ်တာတွေက multiple ရှိရင် ပထမဆုံး match ကိုပဲ ပြန်ပေးမှာပါ
text = '123-4567 456-7890'
text[/\d{3}-\d{4}/]   #=> "123-4567"

slice method ဟာ [] ရဲ့ alias method ဖြစ်ပါတယ်. 

text = 'post-no 123-4567'
text.slice(/\d{3}-\d{4}/)   #=> "123-4567"

#match ဖြစ်တာတွေက multiple ရှိရင် ပထမဆုံး match ကိုပဲ ပြန်ပေးမှာပါ
text = '123-4567 456-7890'
text.slice(/\d{3}-\d{4}/)  #=> "123-4567"

slice! ဆိုရင် match ဖြစ်တဲ့ကောင်ကို ဖြတ်ထုတ်ပေးသွားမှာပါ 

text = 'post-no 123-4567'
text.slice!(/\d{3}-\d{4}/)   #=> "123-4567"
text                         #=> "post-no"

■splitsplit မှာ Regular Expression ကိုထည့်ပေးလိုက်ရင် match ဖြစ်တဲ့ string ကို ဖြတ်ထုတ်ပေးပြီး array ကို ပြန်ပေးမှာပါ. 

text = '123,456-789'
text.split(/,|-/)   #=> ["123", "456", "789"]

■gsub、gsub!အစ parameter က Regular Expression နဲ့ match ဖြစ်တဲ့ string ကို ဒုတိယ parameter က string နဲ့ ပြောင်းပေးမှာပါ 

text = '123,456-789'
text.gsub(/,|-/, ':')    #=> "123:456:789"
​
ဒုတိယ parameter မှာ hash ကိုပေးပြီးတော့ conversion rule ကို ပြောင်းလို့ရပါတယ်.
hash = {',' => ':', '-' => '/'}
text.gsub(/,|-/, hash)   #=> "123:456/789"

gsub! method က string ကြီးတစ်ခုလုံးကို override လုပ်ပြီး change ပေးသွားမှာပါtext = ‘123,456-789’text.gsub!(/,|-/, ‘:’)    #=> “ 123:456:789″text                      #=> “ 123:456:789” 

gsub! method က string ကြီးတစ်ခုလုံးကို override လုပ်ပြီး change ပေးသွားမှာပါ
text = '123,456-789'
text.gsub!(/,|-/, ':')    #=> "123:456:789"
text                      #=> "123:456:789"
ရေးသား ထုတ်ဝေခဲ့ ချိန် စာရေးသူ ကဏ္ဍများ Regular Expression၊ Rubyရဲ့‌ အခြေခံအချက်‌တွေ

Regular Expression အကြောင်း

Ruby မှာ Regular Expression ကိုသုံးမယ်ဆိုရင် သုံးရမယ့် class က Regexp ပါ./ နဲ့စရေးပြီး Regular Expression Object ကို create လုပ်လို့ရပါတယ်. 

regex = /\d{3}-\d{4}/
regex.class     #=> Regexp

Regexp.new() ကိုသုံးပြီးတော့ create လုပ်လို့ရပါတယ်. 

Regexp.new("Ruby")     #=> /Ruby/

%r ကိုသုံးပြီးတော့ create လုပ်လို့ရပါတယ်. 

%r(/Ruby/)     #=>  /\/Ruby\//

Regular Expression =~ string နည်းကိုသုံးမယ်ဆိုရင် match ဖြစ်ရင်  index ကို return ပြန်ပေးပါတယ်.match မဖြစ်ရင် nil ကို return ပြန်ပေးပါတယ်. 

#match ဖြစ်ရင်  index ကို return ပြန်ပေးပါတယ်.
'123-456' =~ /\d{3}-\d{4}/   #=> 0

#match မဖြစ်ရင် nil ကို return ပြန်ပေးပါတယ်
'hello' =~ /\d{3}-\d{4}/    #=> nil
ရေးသား ထုတ်ဝေခဲ့ ချိန် စာရေးသူ ကဏ္ဍများ Regular Expression၊ Rubyရဲ့‌ အခြေခံအချက်‌တွေ

Symbol Feature


symbol ကို  hash နဲ့အတူတူအသုံးပြုတာများပါတယ်.ပုံမှန် hash တစ်ခုက အောက်ပါအတိုင်းပါ 

capital = { 'japan' => 'tokyo', 'myanmar' => 'naypyitaw' }

အဲ့ဒီ hash ရဲ့ key ကို symbol နဲ့ ရေးမယ်ဆို အောက်ပါအတိုင်းရေးရပါမယ်.key ရဲ့ ဘက်ဘက်မှာ : လေးကို ထည့်ပေးလိုက်ရုံပါပဲ.

capital = { :japan => 'tokyo', :myanmar => 'naypyitaw' }
#data ကို ပြန်ထုတ်ကြည့်မယ်
capital[:japan]   #=>  "tokyo"

=> မပါဘဲ ရေးချင်ရင် အောက်ပါအတိုင်း ရေးနိုင်ပါတယ်.key ရဲ့ ညာဘက်မှာ : လေးကို ထည့်ပေးလိုက်ရမှာပါ. 

capital = { japan: 'tokyo', myanmar: 'naypyitaw' }
#data ကို ပြန်ထုတ်ကြည့်မယ်
capital[:japan]   #=>  "tokyo"

key ကိုရော value ကိုရော symbol နဲ့ အောက်ပါအတိုင်း နည်လမ်းနှစ်ခုနဲ့ ရေးနိုင်ပါတယ်.
နံပါတ် 1 နည်းလမ်း(=> ပါတဲ့နည်းလမ်း)key နဲ့ value ရဲ့ ဘက်ဘက်မှာ : လေးကို ထည့်ပေးလိုက်ရုံပါပဲ. 

capital = { :japan => :tokyo, :myanmar => :naypyitaw }
#data ကို ပြန်ထုတ်ကြည့်မယ်
capital[:japan]   #=>  :tokyo

နံပါတ် 2 နည်းလမ်း(=> မပါတဲ့နည်းလမ်း) 

capital = { japan: :tokyo, myanmar: :naypyitaw }
#data ကို ပြန်ထုတ်ကြည့်မယ်
capital[:japan]   #=>  :tokyo
ရေးသား ထုတ်ဝေခဲ့ ချိန် စာရေးသူ ကဏ္ဍများ Rubyရဲ့‌ အခြေခံအချက်‌တွေ၊ Symbol

Symbol အကြောင်း

symbol လုပ်ချင်တဲ့ string ရဲ့ ဘယ်ဘက်‌ဘေးမှာ full column (:) လေးထည့်ပေးလိုက်ရင် symbol ပါပဲ. 

symbol = :name
symbol      #=> :name

％ကိုသုံးပြီး symbol ကို အောက်ပါအတိုင်း create လုပ်နိုင်ပါတယ်.
%s ကို သုံးရမှာပါ 

#
%s!ruby is fun!    #=>  :"ruby is fun"

%s(ruby is fun)    #=>  :"ruby is fun"

symbol array တစ်ခုကို %i ကိုသုံးပြီးတော့ create လုပ်နိုင်ပါတယ်.
အဲ့ဒီအခါကျရင် space ခြားထားတဲ့အလိုက် တစ်ခုချင်းစီက symbol ဖြစ်သွားမှာပါ 

%i(japan myanmar korea)    #=>   [:japan, :myanmar, :korea]

%i မှာဆိုရင် လိုင်းတစ်ကြောင်းဆင်းတာတွေ interpolation တွေက ဒီအတိုင်းပဲ ပြမှာပါ. 

name = 'Alice'
%i(hello\ngood-bye #{name.upcase})   #=>  [:"hello\\ngood-bye", :"\#{name.upcase}"]

%I ကိုသုံးမှ လိုင်းတစ်‌ကြောင်းဆင်းတာတွေ interpolation တွေလုပ်လို့ရပြီး symbol အဖြစ် ပြောင်းပေးမှာပါ 

name = 'Alice'
%I(hello\ngood-bye #{name.upcase})   #=> [:"hello\ngood-bye", :ALICE]
ရေးသား ထုတ်ဝေခဲ့ ချိန် စာရေးသူ ကဏ္ဍများ Rubyရဲ့‌ အခြေခံအချက်‌တွေ၊ Symbol

Hash အကြောင်း

■ hash ကို create လုပ်နည်း

hash ဆိုတာက key value paring ပုံစံဖြစ်ပါတယ်.
hash ကို အောက်ပါအတိုင်း create လုပ်နိုင်ပါတယ်.

{
    'japan' => 'yen',
    'myanmar' => 'kyats'
    'us' => 'dollar'
}

hash တစ်ခုမှာ တူညီတဲ့ key တွေ အများကြီးရှိရင်တော့ နောက်ဆုံး key ကိုပဲ အလုပ်လုပ်မှာပါ. 

{
    'japan' => 'yen',
    'japan' => '円'
}
#=> { "japan" =>  "円"}

■ hash မှ data ကိုယူပုံယူနည်း

currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies['japan']  #=> "yen"

■ hash ထဲသို့ data ထည့်နည်း 

အောက်ပါအတိုင်း data ကို add နိုင်ပါတယ်.
ထည့်ချင်တဲ့ key က hash ထဲမှာ မရှိသေးရင် အသစ် add ပေးပါတယ်  

currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies[thai] = 'bht'
currencies      #=>  { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar', 'thai' => 'baht' }

store method ကိုသုံးပြီးတော့လည်း add လို့ရပါတယ်. 

#hash.store(key, value)
currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies.store('thai', 'baht')
currencies      #=>  { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar', 'thai' => 'baht' }

■ hash မှ data ကို update လုပ်နည်း

ထည့်ချင်တဲ့ key က hash ထဲမှာ ရှိနေရင် override ဖြစ်ပြီးတော့  update လုပ်ပေးပါတယ်  

currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies['japan'] = '円'
currencies      #=>  { 'japan' => '円', 'myanmar' => 'kyats', 'us' => 'dollar', 'thai' => 'baht' }

■ hash မှ data ကို ဖျက်နည်း 

delete method ကိုသုံးပြီးတော့ hash ထဲက data တွေကို ဖျက်နိုင်ပါတယ်. 

currencies = { 'japan' => 'yen', 'myanmar' => 'kyats', 'us' => 'dollar' }
currencies.delete('us')
currencies      #=>  { 'japan' => 'yen', 'myanmar' => 'kyats' }

■ hash ရဲ့ looping

each method ကိုသုံးပြီး loop ပတ်မယ်ဆိုရင် block parameter မှာ key နဲ့ value နှစ်ခုကို ထားပေးရပါမယ်. 

hash = { 'apple' => '300', 'orange' => '200' }
hash.each { |key, value|
    puts "key: #{key}, value: #{value}"
}

#=>     key: apple, value: 300
#       key: orange, value: 200

တကယ်လို့ block parameter မှာ key နဲ့ value နှစ်ခုမထားဘဲ တစ်ခုတည်းပဲ ထားလိုက်မယ်ဆိုရင်တော့ key value တွေက array အနေနဲ့ ၀င်လာမှာပါ. 

hash = { 'apple' => '300', 'orange' => '200' }
hash.each { |fruits|
    key = fruits[0]
    value = fruits[1]
    puts "key: #{key}, value: #{value}"
}

#=>     key: apple, value: 300
#       key: orange, value: 200

■ hash မှ array သို့ data conversion

hash ကနေ array ကို to_a method နဲ့ data conversion လုပ်နိုင်ပါတယ်. to_a method ကိုသုံးလိုက်တဲ့အခါမှာ key value paring တစ်ခုချင်းစီဟာ array တစ်ခုအနေနဲ့ ထွက်လာမှာပါ. array ကြီးတစ်ခုတည်းမှာ key value array လေးတွေ စုထားသလိုမျိုးဖြစ်နေမှာပါ. 

hash = { 'apple' => '300', 'orange' => '200' }
hash.to_a   #=> [["apple", "300"], ["orange", "200"]]
ရေးသား ထုတ်ဝေခဲ့ ချိန် စာရေးသူ ကဏ္ဍများ Hash၊ Rubyရဲ့‌ အခြေခံအချက်‌တွေ

Range

■Range Object ကိုသုံးနည်း

Range ကိုသုံးနည်းက ပုံမှန်အားဖြင့် .. တို့ … တို့ကိုအသုံးပြုသွားမှာပါ.
ဥပမာ
1..5 ဆိုရင် 1 ကနေ 5 အထိ
‘a’..’e’ ဆိုရင် ‌a ကနေ e အထိ
1…5 ဆိုရင် 1 ကနေ 4 အထိ
‘a’…’e’ ဆိုရင် ‌a ကနေ d အထိ

■Case နဲ့ Range ကိုသုံးနည်း

def charge(age)
    case age
    #အသက် 0 နှစ်ကနေ 5 နှစ်အထိ
    when 0..5
        0
    #အသက် 6 နှစ်ကနေ 12 နှစ်အထိ
    when 6..12
        300
    #အသက် 13 နှစ်ကနေ 18 နှစ်အထိ
    when 13..18
        600
    #အခြား
    else
        1000
    end
end

■Array နဲ့ Range ကိုသုံးနည်း

array ရဲ့ index နေရာမှာ index အစား range object ကိုထည့်ပေးလိုက်မယ်ဆိုရင် သတ်မှတ်ထားတဲ့ rage အတိုင်း data တွေကို ထုတ်ယူနိုင်မှာဖြစ်ပါတယ်

a = [1, 2, 3, 4, 5]
a[1..3]    #=> [2, 3, 4]

■looping နဲ့ Range ကိုသုံးနည်း

range ကို array အနေနဲ့ ပြောင်းပြီး loop ပတ်လို့ရပါတယ် 

numbers = (1..4).to_a
sum = 0
numbers.each { |n| sum += n}
sum #=> 10

အောက်ပါအတိုင်း each method နဲ့ loop ပတ်လို့ရပါတယ် 

sum = 0
(1..4).each{ |n| sum += n}
sum #=> 10

အောက်ပါအတိုင်း setp method နဲ့ loop ပတ်လို့ရပါတယ် 

numbers = []
(1..10).step(2){ |n| numbers << n}
numbers #=> [1, 3, 5, 7, 9]

Array အကြောင်း

■ array ကို create လုပ်နည်း

Ruby မှာ array တစ်ခုဆောက်ချင်ရင် လေးထောင့်ကွင်း ([]) ကိုသုံးရပါမယ်.
value တွေအများကြီးထည့်ချင်ရင်တော့ ကြားထဲမှာ comma လေးတွေနဲ့ကြားခံပေးရပါမယ်

fruit = ['apple']
fruits = ['apple', 'banana', 'orange']

data type မတူတာတွေကိုလည်း array တစ်ခုတည်းမှာ သိမ်းထားလို့ရပါတယ်.
အောက်က code မှာ string type နဲ့ number type ကို array တစ်ခုတည်းမှာ သိမ်းထားပုံကို ပြထားပါတယ်

mixing = ['apple', 1, 'banana', 2, 'orange', 3]

array ထဲမှာ နောက်ထပ် array တစ်ခု ထပ်ထည့်လို့ရပါတယ်.

net = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

■ array ထဲက data ကိုထုတ်ယူနည်း

array ထဲက data တွေကို ထုတ်ယူချင်တယ်ဆိုရင် [] နဲ့ index ကိုသုံးရမှာပါ
array ရဲ့ ပထမဆုံး index တန်ဖိုးက 0 ပါ

fruits = ['apple', 'banana', 'orange']
# ပထမဆုံး data ကိုထုတ်ယူတာ
fruits[0]   #=> "apple"
# ဒုတိယ data ကိုထုတ်ယူတာ
fruits[1]   #=> "banana"
# တတိယ data ကိုထုတ်ယူတာ
fruits[2]   #=> "orange"

index တန်ဖိုး 2 ခုသာပေးလိုက်ရင် index ရဲ့ နေရာနဲ့ ထုတ်ယူမယ့် အရေအတွက်ကို setting ချနိုင်မှာပါ

array[position, Length to get]

a = [1, 2, 3, 4, 5]
a[1, 3]      #=> [2, 3, 4]

value_at method ကိုသုံးမယ်ဆိုရင် လိုချင်တဲ့ data တွေရဲ့ index တွေကို အများကြီး setting ချလို့ရပါတယ်.

a = [1, 2, 3, 4, 5]
a.values_at(0, 2, 4)   #=> [1, 3, 5]

Ruby မှာ minus index ကိုသုံးလို့ရပါတယ်.
-1 ဆိုရင် နောက်ဆုံး data, -2 ဆိုရင် နောက်ဆုံးရဲ့ရှေ့က data, အဲ့လိုမျိုးထုတ်ပေးမှာပါ

a = [1, 2, 3]

# နောက်ဆုံးကdataတွေကိုထုတ်
a[-1]    #=> 3
a[-2]    #=> 2

array မှာ last နဲ့ first ဆိုတဲ့ method ရှိပါတယ်. last ဆိုတာကတော့ နောက်ဆုံး data ကိုထုတ်တာပေါ့. parameter မှာ 0 အထက်ကိုပေးလိုက်ရင် နောက်ဆုံးကနေ n အကြိမ်မြောက်အထိ ထုတ်ပေးမှာပါ
first method ကတော့ last နဲ့ပြောင်းပြန် ထိပ်ဆုံးက data ကို ထုတ်ပေးမှာပါ

a = [1, 2, 3]

# last method
a.last     #=> 3
a.last(2)  #=> [2, 3]

# first method
a.first     #=> 1
a.first(2)  #=> [1, 2]

■Array ထဲကို dataထည့်နည်း

push() Method နဲ့ array ထဲကို data ထည့်လို့ရပါတယ်.

a = [1, 2, 3]
a.push(4)    #=> [1, 2, 3, 4]

နောက်တစ်နည်းအနေနဲ့ array index ကို သုံးပြီးတော့ data ထည့်လို့ရပါတယ်.

a = [1, 2, 3]
a[3] = 4	#=> [1, 2, 3, 4]

နောက်တစ်နည်းအနေနဲ့ [<<] ကိုသုံးပြီးတော့ array နောက်ဆုံးအခန်းကနေ data ထပ်ပေါင်းထည့်လို့ရပါတယ်.

a = [1, 2, 3]
a << 4     #=> [1, 2, 3, 4]

■Array ထဲက data ကို ဖျက်နည်း

array ထဲက data ကိုဖျက်ချင်ရင် delete() Method ကိုသုံးလို့ရပါတယ်.

a = [1, 2, 3]
a.delete(1)     #=> [2, 3]

index အလိုက်ဖျက်ချင်ရင်တော့ delete_at() Method ကိုသုံးနိုင်ပါတယ်.

a = [1, 2, 3]
a.delete_at(1)   #=> [1, 3]

#မရှိတဲ့ index ပေးပြီး delete လုပ်ရင် nil ကို return ပြန်
a = [1, 2, 3]
a.delete_at(5)     #=> nil

■array နဲ့ block သုံးပုံသုံးနည်း

each method ကိုသာသုံးမယ်ဆိုရင် array ရဲ့ index တွေကို သိနိုင်မှာမဟုတ်ပါဘူး.ဒါပေမယ့် each_with_index ကိုသုံးမယ်ဆိုရင်တော့ index ကိုသိနိုင်မှာဖြစ်ပါတယ်.

countries = ['japan', 'myanmar', 'america']
countries.each_with_index { |country, index| puts "#{index}:  #{country}" }
#=> 0: japan
#   1: myanmar
#   2: america

အလားတူပုံစံမျိုးကို map method နဲ့တခြား method တွေမှာ သုံးမယ်ဆိုရင်တော့ with_index ကိုသုံးနိုင်ပါတယ်. 

countries = ['japan', 'myanmar', 'america']
countries.map.with_index { |country, index| "#{index}:  #{country}" }
#=> ["0:  japan", "1:  myanmar", "2:  america"]

countries.delete_if.with_index{ |country, index| country.include?('y') && index.odd? }
#=> ["japan", "america"]

with_index ဟာ ဘာဖြစ်လို့ map method, each method, delete_if method တွေကနေ ခေါ်သုံးလို့ရနေရလဲဆိုတော့ with_index ဟာ Enumerator class ရဲ့ instance method တစ်ခုဖြစ်ပါတယ်. map method, each method, delete_if method တွေမှာ {} ကိုဖြုတ်ပြီး method call လုပ်မယ်ဆို Enumerator object ကို ပြန်ပေးမှာဖြစ်ပါတယ်.အဲ့ဒါကြောင့်မို့လို့ ခုနက looping တွေကနေ with_index ကို ခေါ်သုံးလို့ရနေတာဖြစ်ပါတယ်. 

countries.map       #=> #<Enumerator: ["japan", "myanmar", "america"]:map>
countries.each      #=> #<Enumerator: ["japan", "myanmar", "america"]:each>
countries.delete_if #=> #<Enumerator: ["japan", "myanmar", "america"]:delete_if>

2 dimensional array အတွက်ဆိုရင်တော့ နည်းနည်းကွာပါလိမ့်မယ်. array ကို block parameter အနေနဲ့ပေးလိုက်ရင် အောက်ပါအတိုင်း row col အနေနဲ့  ပြန်ထုတ်ရပါလိမ့်မယ်. 

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each do |dimension| 
    length = dimension[0]
    width = dimension[1]
    puts "length: #{length}, width: #{width}"
end
#=>     length: 10, width: 20
#=>     length: 30, width: 40
#=>     length: 50, width: 60

တကယ်လို့သာ block ရဲ့ parameter မှာ parameter 2 ခုသာပေးလိုက်ရင် row col  အနေနဲ့  ချက်ချင်းတန်းသုံးလို့ရမှာဖြစ်ပါတယ်. 

dimensions.each do |length, width|
    puts "length: #{length}, width: #{width}"
end
#=>     length: 10, width: 20
#=>     length: 30, width: 40
#=>     length: 50, width: 60

2 dimensional array မှာ index value ကို သိချင်တယ်ဆိုရင် block parameter မှာ parameter 2 ခု ပေးရမှာဖြစ်ပါတယ်.ရှေ့ parameter မှာ array ၀င်လာပြီး နောက် parameter မှာက index တန်ဖိုး ၀င်လာမှာ ဖြစ်ပါတယ်. 

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each_with_index do |dimension, i| 
    length = dimension[0]
    width = dimension[1]
    puts "length: #{length}, width: #{width}, index: #{i}"
end
#=>     length: 10, width: 20, index: 0
#=>     length: 30, width: 40, index: 1
#=>     length: 50, width: 60, index: 2

နောက်တစ်နည်း အနေနဲ့ block parameter variable ကို သုံးချင်တယ်ဆိုရင်တော့ variable တွေကို () ကွင်းခတ်ပေးရမှာဖြစ်ပါတယ်. 

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each_with_index do |(length, width), i|
    puts "length: #{length}, width: #{width}, index: #{i}"
end
#=>     length: 10, width: 20, index: 0
#=>     length: 30, width: 40, index: 1
#=>     length: 50, width: 60, index: 2

block အတွင်းထဲက variable က အပြင်မှာ သုံးလို့မရပါဘူး. 

sum = 0
numbers = [10, 20]
numbers.each do |num; sum|
    sum = 1
    sum += num
    puts "In Block: #{sum}"
end
puts "Outer Block: #{sum}"

#=>     In Block: 11
#=>     In Block: 21
#=>     Outer Block: 0
ရေးသား ထုတ်ဝေခဲ့ ချိန် စာရေးသူ ကဏ္ဍများ Array၊ Rubyရဲ့‌ အခြေခံအချက်‌တွေ

ညီမညီစစ်တဲ့ Method များ

Ruby မှာ ညီမညီစစ်ချင်ရင် သုံးလို့ရတဲ့ method နှစ်ခု equal? နဲ့ eql? ဆိုပြီး 2 ခုရှိပါတယ်

■equal?

equal? methodကို instance တူလား မတူဘူးလားစစ်တဲ့နေရာမှာသုံးပါတယ်

a = 'abc'
b = 'abc'
a.equal?(b)         #=> false

c = a
a.equal?(c)         #=> true

■eql?

eql? method ကို string ထဲက data က တူလားလို့စစ်တဲ့နေရာမှာသုံးပါတယ်.
hash မှာဆိုရင် key တူလားမတူလားဆိုတာကို စစ်တဲ့နေရာမှာသုံးပါတယ်

# string ထဲက data က တူလားလို့စစ်
a = 'abc'
b = 'abc'
a.eql?(b)        #=> true

# hash မှာဆိုရင် key တူလားမတူလားဆိုတာကို စစ်
h = { 1 => 'Integer', 1.0 => 'Float' }
h[1]    #=>  "Integer"
h[1.0]  #=>  "Float"

# key မတူပါဘူး
1.eql?(1.0)  #=> false
ရေးသား ထုတ်ဝေခဲ့ ချိန် စာရေးသူ ကဏ္ဍများ Method၊ Rubyရဲ့‌ အခြေခံအချက်‌တွေ

Method Name ရဲ့ Question Mark​(?) နဲ့ Exclamation Mark(!)

■question mark​(?)

(?) နဲ့ဆုံးတဲ့ method တွေဟာဆိုရင် true or false တစ်ခုခု return ပြန်မယ်ဆိုတဲ့ method တွေဖြစ်ပါတယ်

# empty သာဆိုရင် true, အဲ့လိုမဟုတ်ဘူးဆိုရင် false
''.empty?        #=> true
'abcd'.empty?    #=> false

# object က nil ဆိုရင် true, အဲ့လိုမဟုတ်ဘူးဆိုရင် false
@foo.nil?     #=> true
@foo = 'foo'
@foo.nil?     #=> false

# မကိန်းလား
1.odd?    #=> true

# စုံကိန်းလား

array = [3, 1, 5, 2, 4]

a = array.sort
puts a       #=> [1, 2, 3, 4, 5]
puts array   #=> [3, 1, 5, 2, 4]

b = array.sort!
puts b      #=> [1, 2, 3, 4, 5]
puts array  #=> [1, 2, 3, 4, 5]
