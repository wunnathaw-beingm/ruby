Range

■Range Object ကိုသုံးနည်း

Range ကိုသုံးနည်းက ပုံမှန်အားဖြင့် .. တို့ … တို့ကိုအသုံးပြုသွားမှာပါ.
ဥပမာ
1..5 ဆိုရင် 1 ကနေ 5 အထိ
‘a’..’e’ ဆိုရင် ‌a ကနေ e အထိ
1…5 ဆိုရင် 1 ကနေ 4 အထိ
‘a’…’e’ ဆိုရင် ‌a ကနေ d အထိ

■Case နဲ့ Range ကိုသုံးနည်း

def charge(age)
    case age
    #အသက် 0 နှစ်ကနေ 5 နှစ်အထိ
    when 0..5
        0
    #အသက် 6 နှစ်ကနေ 12 နှစ်အထိ
    when 6..12
        300
    #အသက် 13 နှစ်ကနေ 18 နှစ်အထိ
    when 13..18
        600
    #အခြား
    else
        1000
    end
end

■Array နဲ့ Range ကိုသုံးနည်း

array ရဲ့ index နေရာမှာ index အစား range object ကိုထည့်ပေးလိုက်မယ်ဆိုရင် သတ်မှတ်ထားတဲ့ rage အတိုင်း data တွေကို ထုတ်ယူနိုင်မှာဖြစ်ပါတယ်

a = [1, 2, 3, 4, 5]
a[1..3]    #=> [2, 3, 4]

■looping နဲ့ Range ကိုသုံးနည်း

range ကို array အနေနဲ့ ပြောင်းပြီး loop ပတ်လို့ရပါတယ် 

numbers = (1..4).to_a
sum = 0
numbers.each { |n| sum += n}
sum #=> 10

အောက်ပါအတိုင်း each method နဲ့ loop ပတ်လို့ရပါတယ် 

sum = 0
(1..4).each{ |n| sum += n}
sum #=> 10

အောက်ပါအတိုင်း setp method နဲ့ loop ပတ်လို့ရပါတယ် 

numbers = []
(1..10).step(2){ |n| numbers << n}
numbers #=> [1, 3, 5, 7, 9]

Looping

■times method

အကြိမ်အရေအတွက် အတိအကျနဲ့ looping ပတ်ချင်တယ်ဆိုရင် integer class ရဲ့  times method ကို သုံးနိုင်ပါတယ်. 

sum = 0
#n ထဲကို 0 ကနေ 9 အထိ တန်ဖိုးတွေ ၀င်ပါမယ်
10.times { |n|
    sum += n
    puts n
}
puts sum      #=> 45

■ upto method

n ကနေ m အထိ အပေါ်ကိုတစ်ဆင့်ချင်းစီ looping ပတ်ချင်ရင်တော့  upto method ကို သုံးလို့ရပါတယ်.

1.upto(5) { |n| puts "count: #{n}"}
#=>     count: 1
#=>     count: 2
#=>     count: 3
#=>     count: 4
#=>     count: 5

■ downto method

n ကနေ m အထိ အောက်ကိုတစ်ဆင့်ချင်းစီ looping ပတ်ချင်ရင်တော့  downto method ကိုသုံးလို့ရပါတယ်. 

5.downto(1) { |n| puts "count: #{n}"}
#=>     count: 5
#=>     count: 4
#=>     count: 3
#=>     count: 2
#=>     count: 1

■ step method

n ကနေ m အထိကို တစ်ကြိမ်ချင်းစီမဟုတ်ဘဲ x အကြိမ်အရေအတွက်အလိုက် တိုးပြီးတော့ looping ပတ်ချင်ရင် step method ကို သုံးနိင်ပါတယ်. 

1.step(10, 2) { |n| puts "count: #{n}"}
#=>     count: 1
#=>     count: 3
#=>     count: 5
#=>     count: 7
#=>     count: 9

■ while looping

ပေးလိုက်တဲ့ statement က true ဖြစ်နေသရွေ့ looping ကိုပတ်နေမှာ ဖြစ်ပါတယ်. 

n = 0
while n < 5
    puts n += 1
end
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

statement နောက်မှာ do ထည့်ပေးလိုက်ရင် တစ်ကြောင်းတည်းနဲ့ ရေးနိုင်မှာ ဖြစ်ပါတယ်. 

n = 0
while n < 5 do puts n += 1 end
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

do end မထည့်ချင်ရင် while statement ကို နောက်ကိုပို့ရေးလိုက်ရင်ရပါတယ်. 

n = 0
puts n += 1 while n < 5
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

ဘယ်လိုပဲဖြစ်ဖြစ် ပထမဆုံးတစ်ကြိမ်ကို မဖြစ်မနေ ၀င်ချင်တယ်ဆိုရင် begin end ကို တွဲသုံးလို့ရပါတယ်. 

n = 0
begin
    n = 1
end while false
puts n    #=> 1

■ loop

■ next

Array အကြောင်း

■ array ကို create လုပ်နည်း

Ruby မှာ array တစ်ခုဆောက်ချင်ရင် လေးထောင့်ကွင်း ([]) ကိုသုံးရပါမယ်.
value တွေအများကြီးထည့်ချင်ရင်တော့ ကြားထဲမှာ comma လေးတွေနဲ့ကြားခံပေးရပါမယ်

fruit = ['apple']
fruits = ['apple', 'banana', 'orange']

data type မတူတာတွေကိုလည်း array တစ်ခုတည်းမှာ သိမ်းထားလို့ရပါတယ်.
အောက်က code မှာ string type နဲ့ number type ကို array တစ်ခုတည်းမှာ သိမ်းထားပုံကို ပြထားပါတယ်

mixing = ['apple', 1, 'banana', 2, 'orange', 3]

array ထဲမှာ နောက်ထပ် array တစ်ခု ထပ်ထည့်လို့ရပါတယ်.

net = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

■ array ထဲက data ကိုထုတ်ယူနည်း

array ထဲက data တွေကို ထုတ်ယူချင်တယ်ဆိုရင် [] နဲ့ index ကိုသုံးရမှာပါ
array ရဲ့ ပထမဆုံး index တန်ဖိုးက 0 ပါ

fruits = ['apple', 'banana', 'orange']
# ပထမဆုံး data ကိုထုတ်ယူတာ
fruits[0]   #=> "apple"
# ဒုတိယ data ကိုထုတ်ယူတာ
fruits[1]   #=> "banana"
# တတိယ data ကိုထုတ်ယူတာ
fruits[2]   #=> "orange"

index တန်ဖိုး 2 ခုသာပေးလိုက်ရင် index ရဲ့ နေရာနဲ့ ထုတ်ယူမယ့် အရေအတွက်ကို setting ချနိုင်မှာပါ

array[position, Length to get]

a = [1, 2, 3, 4, 5]
a[1, 3]      #=> [2, 3, 4]

value_at method ကိုသုံးမယ်ဆိုရင် လိုချင်တဲ့ data တွေရဲ့ index တွေကို အများကြီး setting ချလို့ရပါတယ်.

a = [1, 2, 3, 4, 5]
a.values_at(0, 2, 4)   #=> [1, 3, 5]

Ruby မှာ minus index ကိုသုံးလို့ရပါတယ်.
-1 ဆိုရင် နောက်ဆုံး data, -2 ဆိုရင် နောက်ဆုံးရဲ့ရှေ့က data, အဲ့လိုမျိုးထုတ်ပေးမှာပါ

a = [1, 2, 3]

# နောက်ဆုံးကdataတွေကိုထုတ်
a[-1]    #=> 3
a[-2]    #=> 2

array မှာ last နဲ့ first ဆိုတဲ့ method ရှိပါတယ်. last ဆိုတာကတော့ နောက်ဆုံး data ကိုထုတ်တာပေါ့. parameter မှာ 0 အထက်ကိုပေးလိုက်ရင် နောက်ဆုံးကနေ n အကြိမ်မြောက်အထိ ထုတ်ပေးမှာပါ
first method ကတော့ last နဲ့ပြောင်းပြန် ထိပ်ဆုံးက data ကို ထုတ်ပေးမှာပါ

a = [1, 2, 3]

# last method
a.last     #=> 3
a.last(2)  #=> [2, 3]

# first method
a.first     #=> 1
a.first(2)  #=> [1, 2]

■Array ထဲကို dataထည့်နည်း

push() Method နဲ့ array ထဲကို data ထည့်လို့ရပါတယ်.

a = [1, 2, 3]
a.push(4)    #=> [1, 2, 3, 4]

နောက်တစ်နည်းအနေနဲ့ array index ကို သုံးပြီးတော့ data ထည့်လို့ရပါတယ်.

a = [1, 2, 3]
a[3] = 4	#=> [1, 2, 3, 4]

နောက်တစ်နည်းအနေနဲ့ [<<] ကိုသုံးပြီးတော့ array နောက်ဆုံးအခန်းကနေ data ထပ်ပေါင်းထည့်လို့ရပါတယ်.

a = [1, 2, 3]
a << 4     #=> [1, 2, 3, 4]

■Array ထဲက data ကို ဖျက်နည်း

array ထဲက data ကိုဖျက်ချင်ရင် delete() Method ကိုသုံးလို့ရပါတယ်.

a = [1, 2, 3]
a.delete(1)     #=> [2, 3]

index အလိုက်ဖျက်ချင်ရင်တော့ delete_at() Method ကိုသုံးနိုင်ပါတယ်.

a = [1, 2, 3]
a.delete_at(1)   #=> [1, 3]

#မရှိတဲ့ index ပေးပြီး delete လုပ်ရင် nil ကို return ပြန်
a = [1, 2, 3]
a.delete_at(5)     #=> nil

■array နဲ့ block သုံးပုံသုံးနည်း

each method ကိုသာသုံးမယ်ဆိုရင် array ရဲ့ index တွေကို သိနိုင်မှာမဟုတ်ပါဘူး.ဒါပေမယ့် each_with_index ကိုသုံးမယ်ဆိုရင်တော့ index ကိုသိနိုင်မှာဖြစ်ပါတယ်.

countries = ['japan', 'myanmar', 'america']
countries.each_with_index { |country, index| puts "#{index}:  #{country}" }
#=> 0: japan
#   1: myanmar
#   2: america

အလားတူပုံစံမျိုးကို map method နဲ့တခြား method တွေမှာ သုံးမယ်ဆိုရင်တော့ with_index ကိုသုံးနိုင်ပါတယ်. 

countries = ['japan', 'myanmar', 'america']
countries.map.with_index { |country, index| "#{index}:  #{country}" }
#=> ["0:  japan", "1:  myanmar", "2:  america"]

countries.delete_if.with_index{ |country, index| country.include?('y') && index.odd? }
#=> ["japan", "america"]

with_index ဟာ ဘာဖြစ်လို့ map method, each method, delete_if method တွေကနေ ခေါ်သုံးလို့ရနေရလဲဆိုတော့ with_index ဟာ Enumerator class ရဲ့ instance method တစ်ခုဖြစ်ပါတယ်. map method, each method, delete_if method တွေမှာ {} ကိုဖြုတ်ပြီး method call လုပ်မယ်ဆို Enumerator object ကို ပြန်ပေးမှာဖြစ်ပါတယ်.အဲ့ဒါကြောင့်မို့လို့ ခုနက looping တွေကနေ with_index ကို ခေါ်သုံးလို့ရနေတာဖြစ်ပါတယ်. 

countries.map       #=> #<Enumerator: ["japan", "myanmar", "america"]:map>
countries.each      #=> #<Enumerator: ["japan", "myanmar", "america"]:each>
countries.delete_if #=> #<Enumerator: ["japan", "myanmar", "america"]:delete_if>

2 dimensional array အတွက်ဆိုရင်တော့ နည်းနည်းကွာပါလိမ့်မယ်. array ကို block parameter အနေနဲ့ပေးလိုက်ရင် အောက်ပါအတိုင်း row col အနေနဲ့  ပြန်ထုတ်ရပါလိမ့်မယ်. 

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each do |dimension| 
    length = dimension[0]
    width = dimension[1]
    puts "length: #{length}, width: #{width}"
end
#=>     length: 10, width: 20
#=>     length: 30, width: 40
#=>     length: 50, width: 60

တကယ်လို့သာ block ရဲ့ parameter မှာ parameter 2 ခုသာပေးလိုက်ရင် row col  အနေနဲ့  ချက်ချင်းတန်းသုံးလို့ရမှာဖြစ်ပါတယ်. 

dimensions.each do |length, width|
    puts "length: #{length}, width: #{width}"
end
#=>     length: 10, width: 20
#=>     length: 30, width: 40
#=>     length: 50, width: 60

2 dimensional array မှာ index value ကို သိချင်တယ်ဆိုရင် block parameter မှာ parameter 2 ခု ပေးရမှာဖြစ်ပါတယ်.ရှေ့ parameter မှာ array ၀င်လာပြီး နောက် parameter မှာက index တန်ဖိုး ၀င်လာမှာ ဖြစ်ပါတယ်. 

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each_with_index do |dimension, i| 
    length = dimension[0]
    width = dimension[1]
    puts "length: #{length}, width: #{width}, index: #{i}"
end
#=>     length: 10, width: 20, index: 0
#=>     length: 30, width: 40, index: 1
#=>     length: 50, width: 60, index: 2

နောက်တစ်နည်း အနေနဲ့ block parameter variable ကို သုံးချင်တယ်ဆိုရင်တော့ variable တွေကို () ကွင်းခတ်ပေးရမှာဖြစ်ပါတယ်. 

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each_with_index do |(length, width), i|
    puts "length: #{length}, width: #{width}, index: #{i}"
end
#=>     length: 10, width: 20, index: 0
#=>     length: 30, width: 40, index: 1
#=>     length: 50, width: 60, index: 2

block အတွင်းထဲက variable က အပြင်မှာ သုံးလို့မရပါဘူး. 

sum = 0
numbers = [10, 20]
numbers.each do |num; sum|
    sum = 1
    sum += num
    puts "In Block: #{sum}"
end
puts "Outer Block: #{sum}"

#=>     In Block: 11
#=>     In Block: 21
#=>     Outer Block: 0
