■ အကုန်လုံးက object

Ruby ဟာ OOP Language တစ်ခုဖြစ်ပါတယ်.
Ruby မှာ အကုန်လုံးဟာ object တွေဖြစ်ကြပါတယ်။ string, array, number နဲ့ nil တွေ စသဖြင့်အကုန်လုံးဟာ object တွေဖြစ်ပါတယ်။
number, nil, true တွေ false တွေကနေ method တွေကို ခေါ်နိုင် ပါတယ်။

# String
 'abc'.to_s        #=> "abc"

# Number
 100.to_s        #=> "100"

# nil
 nil.to_s        #=> "nil"

# true
 true.to_s       #=> "true"

　　　

■ Ruby ရဲ့ လိုက်လျောညီထွေနိုင်မှု ( flexibility )

Ruby ဟာ အရမ်းကို flexible ဖြစ်တဲ့ Language တစ်ခုလို့ဆိုကြပါတယ်။
ဥပမာပြောရမယ်ဆိုရင် အပေါင်းလက္ခဏာနဲ့ ပေါင်းတာကိုလုပ်နေကြနေပေမယ့် plus ဆိုတဲ့ စာလုံးနဲ့ ပေါင်းတာကို လုပ်မယ်ဆိုလည်း လုပ်နိုင်ပါတယ်။ အဲ့လိုသာလုပ်ချင်တယ်ဆိုရင် Ruby ရဲ့ build-in class ဖြစ်တဲ့ numeric class မှာ အောက်ပါအတိုင်း method ကို သွားထည့်နိုင်ပါတယ်။

class Numeric
	def plus(x)
      self.+(x)
    end
end

y = 5.plus(4) 	#=> 9
ရေးသား ထုတ်ဝေခဲ့ ချိန် စာရေးသူ ကဏ္ဍများ Rubyရဲ့‌ အခြေခံအချက်‌တွေ
