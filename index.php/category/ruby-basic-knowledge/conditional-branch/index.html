三項演算子の使い方

三項演算子(条件演算子）は条件式の結果によって異なる値を演算子です。書式は下記の通りです。

条件式 ? 真の時の値 : 偽の時の値

具体的な使い方は下記の通りです。

mark = 80
mark > 60 ? "合格" : "不合格"  #> "合格"

次のように、条件分岐した結果を変数に代入することも可能です。

mark = 80
result = mark > 60 ? "合格" : "不合格"
result      #=> "合格"

case文の使い方

複数の条件を指定する場合は、elsifを重ねるよりもcase文で書いたほうがシンプルになります。case文の構文は次のとおりです。

case  対象オブジェクト
when 値1
  値1と一致する場合の処理
when 値2
  値2と一致する場合の処理
when 値3
  値3と一致する場合の処理
else
  どの値にも一致しない場合の処理
end

具体的の書き方は下記の通りです。

country = 'italy'

#case文の使い方
case country
  when 'japan'
  'こんにちは！'
  when 'us'
   'Hello!'
  when 'italy'
  'ciao!'
else
  '???'
end         #=>"ciao!"

when節に複数の値を指定し、どれかに一致する処理を実行することもできます。

country = 'アメリカ'

#when節に複数の値を指定する
case country
  when 'japan', '日本'
  'こんにちは！'
  when 'us', 'アメリカ'
   'Hello!'
  when 'italy', 'イアリア'
  'ciao!'
else
  '???'
end         #=>"Hello!"

unless文の使い方

if文では条件式が真かどうかで処理を分けますが、unlessはifと反対の意味を持っています。unless文は条件式が偽の場合の処理を記述するのに使われます。書式は次の通りです。

unless 条件式 
  #処理A
end

具体的には次のように使用します。

status = 'error'
unless status == 'ok'
  '何か異常があります'
end                #=> "何か異常があります"

elseを使って条件が偽でなかった場合の処理を書くこともできます。

status = 'ok'
unless status == 'ok'
  '何か異常があります'
else
  '正常です'
end        #=> "正常です"

if文の使い方

Rubyのif文の基本構文は次のように書きます。

if 条件式1
  #処理A
elsif 条件式2
  #処理B
else
  #処理C
end

具体的なサンプルは下記の通りです。

a = 15

if a > 30 
  puts "a is greater than 30"
elsif a > 20
  puts "a is greater than 20"
elseif a > 10
  puts "a is greater than 10"
else
  puts "a is less than 10"
end               #=> "a is greater than 10"

if文が戻り値を返すという性質を利用して、次のようにif文の戻り値を変数に代入することもできます。

country = 'italy'

#if文の戻り値を変数に代入する
greeting =
  if country == 'japan'
    'こんにちは！'
  elsif country == 'us'
    'Hello!'
  elsif country == 'italy'
    'ciao!'
  else
    '???'
  end

greeting			#=>  "ciao!"

Rubyにおける真偽値

if分やwhile分での条件判定に使う「真」と「偽」の扱いはプログラム言語により様々です。Rubyでは真偽値は次のようなルールを持っています。

・falseとnilであれば「偽」
・それ以外が「真」
として扱われます。

#nilは「偽」？
if nil
  puts "nil is true"
else
  puts "nil is false"      #=> "nil is false"
end

#falseは「偽」?
if false
  puts "false is true"
else
  puts "false is false"		#=> "false is false"
end

#数字の1は「真」?
if 1 
  puts "1 is true" 
else
  puts "1 is false"         #=> "1 is true"
end
