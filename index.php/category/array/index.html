範囲(Range)

■範囲オブジェクトの指定方法

範囲オブジェクトは次のように..また…を使って作成します。
具体的には次のように作成します。
1..5 # 1から5まで
‘a’..’e’ #aからeまで
1…5　 #1から5まで
‘a’…’e’ #aからeまで

■case文で使う範囲

def charge(age)
    case age
    #0歳から5歳
    when 0..5
        0
    #6歳から12歳
    when 6..12
        300
    #13歳から18歳
    when 13..18
        600
    #それ以外
    else
        1000
    end
end

■配列で使う範囲

配列に対して添え字の代わりに範囲オブジェクトを渡すと、指定した範囲の要素を取得することができます。

a = [1, 2, 3, 4, 5]
a[1..3]    #=> [2, 3, 4]

■繰り返し処理で使う範囲

範囲オブジェクトを配列に変換して繰り返し処理を行うことができます。

numbers = (1..4).to_a
sum = 0
numbers.each { |n| sum += n}
sum #=> 10

範囲オブジェクトに対して直接eachメソッドを利用して繰り返し処理を行うこともできます。

sum = 0
(1..4).each{ |n| sum += n}
sum #=> 10

stepメソッドを利用して値を増やす間隔を指定できます。

numbers = []
(1..10).step(2) { |n| number << n }
numbers   #=> [1, 3, 5, 7, 9]

繰り返し処理

■timesメソッド

n回処理を繰り返したいと言う場合は、integerクラスのtimesメソッドを利用すると便利です。

sum = 0
#処理を10回繰り返す。nには0から9が入る
10.times { |n|
  sum += n
  }
 puts sum    #=> 45

■uptoメソッド

nからmまで数値を１つずつ増やしながら処理したい場合は、uptoメソッドを利用すると便利です。

1.upto(5) { |n| puts "count: #{n}" }
#=>     count: 1
#=>     count: 2
#=>     count: 3
#=>     count: 4
#=>     count: 5

■downtoメソッド

nからmまで数値を減らしていきたい場合は、downtoメソッドを利用すると便利です。

5.downto(1) { |n| puts "count: #{n}" }
#=>     count: 5
#=>     count: 4
#=>     count: 3
#=>     count: 2
#=>     count: 1

■stepメソッド

nからmまで数値をx個ずつ増やしていきたい場合は、stepメソッドを利用します。

1.step(10, 2) { |n| puts "count: #{n}"}
#=>     count: 1
#=>     count: 3
#=>     count: 5
#=>     count: 7
#=>     count: 9

■while文

while文は指定した条件が真である間、処理を繰り返します。

n = 0
while n < 5
    puts n += 1
end
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

条件式の後ろにdoを入れると1行で書けます。

n = 0
while n < 5 do puts n += 1 end
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

1行で書くのであれば修飾子としてwhile文を後ろに置いた方がスッキリ書けます。

n = 0
puts n += 1 while n < 5
#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

どんな条件でも最低1回は実行したい場合は、begin…endで囲んでからwhileを書きます。

n = 0
begin
    n = 1
end while false
puts n    #=> 1

■until文

until文はwhile文と反対で、条件が偽である間繰り返し処理を行います。

num = 10
until num == 15 do
    puts num
    num += 1
end
#=>     10
#=>     11
#=>     12
#=>     13
#=>     14

■for文

for num in 1..5 do
  p num
end

#=>     1
#=>     2
#=>     3
#=>     4
#=>     5

■loopメソッド

loopメソッドを使うと無限ループを作ることができます。無限ループから脱出する場合はbreakを使います。

num =0
loop{
  p num
  num+=2
  if num >= 10
    break
  end
}
  
#=>     0
#=>     2
#=>     4
#=>     6
#=>     8

■next

繰り返し処理を途中で中断し、次の繰り返し処理に進める場合はnextを使います。

[1,2,3,4].each do |num|
  next if num == 1
  p num
end

#=>   2
#=>   3
#=>   4

■redo

繰り返し処理をやりなおしたい場合はredoを使います。

[0,1,2,3,4].each do |num|
  num += 1
  p num
  redo if num == 1
end

#=>   1
#=>   2
#=>   2
#=>   3
#=>   4
#=>   5

配列のさまざまな操作

■配列を作成する方法

Rubyにおいて配列を作成する場合は、括弧（[])を使用します。値が複数ある配列は値をカンマ区切りで記述します。

fruit = ['apple']
fruits = ['apple', 'banana', 'orange']

異なるデータ型を格納することもできます。次は数値と文字列を混在させた配列です。

mixing = ['apple', 1, 'banana', 2, 'orange', 3]

配列の中に配列を含めることもできます。

net = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

■配列から要素を取得する方法

配列の各要素を取得する場合は、[]と添え字を使います。最初の要素の添え字は0です。

fruits = ['apple', 'banana', 'orange']
#1つ目の要素を取得する
fruits[0]   #=> "apple"
#2つ目の要素を取得する
fruits[1]   #=> "banana"
#3つ目の要素を取得する
fruits[2]   #=> "orange"

添え字を2つ使うと、添え字の位置と取得する長さを指定することができます。

配列[位置, 取得する長さ]

a = [1, 2, 3, 4, 5]
a[1, 3]      #=> [2, 3, 4]

values_atメソッドを使うと、取得したい要素の添え字を複数指定できます。

a = [1, 2, 3, 4, 5]
a.values_at(0, 2, 4)   #=> [1, 3, 5]

Rubyでは添え字を負の値が使えます。-1は最後の要素、-2は最後から2番目の要素というように指定できます。

a = [1, 2, 3]

#最後の要素
a[-1]    #=> 3
a[-2]    #=> 2

配列にlastとfirstというメソッドがあります。lastは配列の最後の要素を取得できます。引数に0以上の数値を渡すと、最後のn個の要素を取得できます。firstはlastの反対で先頭の要素を取得するメソッドです。

a = [1, 2, 3]

#lastメソッド
a.last   #=> 3
a.last(2)  #=> [2, 3]

#firstメソッド
a.first  #=> 1
a.first(2)  #=> [1, 2]

■配列に要素を追加する方法

配列に要素を追加する場合、push()メソッドが使えます。

a = [1, 2, 3]
a.push(4)    #=> [1, 2, 3, 4]

また、添え字を指定して、要素を追加することもできます。

a = [1, 2, 3]
a[3] = 4	#=> [1, 2, 3, 4]

また、[<<]記号を使用して、配列の末尾に要素を追加できます。

a = [1, 2, 3]
a << 4     #=> [1, 2, 3, 4]

■ 配列の要素を削除する方法

配列の要素を削除する場合、delete()メソッドが使えます。

a = [1, 2, 3]
a.delete(1)     #=> [2, 3]

指定位置の要素を削除する場合、delete_at()メソッドを使えます。

a = [1, 2, 3]
a.delete_at(1)   #=> [1, 3]

#存在しない添え字を指定するとnilが返る
a = [1, 2, 3]
a.delete_at(5)     #=> nil

■ 配列とブロックの使い方

eachメソッドを使って繰り返し処理を行うと何番目の要素を処理しているのか、ブロック内で判別できません。each_with_indexメソッドを使うと添え字を判別できます。

countries = ['japan', 'myanmar', 'america']
countries.each_with_index { |country, index| puts "#{index}:  #{country}" }
#=> 0: japan
#   1: myanmar
#   2: america

mapメソッドで繰り返し処理をしつつ、添え字も同時に取得したいときは
mapメソッドとwith_indexメソッドを組み合わせて使います。

countries = ['japan', 'myanmar', 'america']
countries.map.with_index { |country, index| "#{index}:  #{country}" }
#=> ["0:  japan", "1:  myanmar", "2:  america"]

countries.delete_if.with_index{ |country, index| country.include?('y') && index.odd? }
#=> ["japan", "america"]

with_methodはEnumeratorクラスのインスタンスメソッドです。map, each, delete_ifメソッドなど繰り返し処理を行うとメソッドのブロックを省略して呼び出すと、Enumeratorオブジェクトを返します。

countries.map       #=> #<Enumerator: ["japan", "myanmar", "america"]:map>
countries.each      #=> #<Enumerator: ["japan", "myanmar", "america"]:each>
countries.delete_if #=> #<Enumerator: ["japan", "myanmar", "america"]:delete_if>

配列の配列に対して、eachメソッドなどで繰り返し処理を実行すると、配列がブロック引数に渡ってきます。

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each do |dimension| 
    length = dimension[0]
    width = dimension[1]
    puts "length: #{length}, width: #{width}"
end
#=>     length: 10, width: 20
#=>     length: 30, width: 40
#=>     length: 50, width: 60

しかし、ブロック引数の数を2個にすると、値を別々に受け取ることができます。

dimensions.each do |length, width|
    puts "length: #{length}, width: #{width}"
end
#=>     length: 10, width: 20
#=>     length: 30, width: 40
#=>     length: 50, width: 60

each_with_indexを使ってブロック引数を2個にすると、第1引数で縦と横の値を配列として取得できます。

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each_with_index do |dimension, i| 
    length = dimension[0]
    width = dimension[1]
    puts "length: #{length}, width: #{width}, index: #{i}"
end
#=>     length: 10, width: 20, index: 0
#=>     length: 30, width: 40, index: 1
#=>     length: 50, width: 60, index: 2

配列の要素を受け取るブロック引数を()で囲むと、配列の要素を別々の引数として受け取ることができます。

dimensions = [
    [10, 20],
    [30, 40],
    [50, 60],
]

dimensions.each_with_index do |(length, width), i|
    puts "length: #{length}, width: #{width}, index: #{i}"
end
#=>     length: 10, width: 20, index: 0
#=>     length: 30, width: 40, index: 1
#=>     length: 50, width: 60, index: 2

ブロック引数を；で区切り、続けて変数を宣言すると、ブロック内でのみ有効な独立したローカル変数を宣言することができます。

sum = 0
numbers = [10, 20]
numbers.each do |num; sum|
    sum = 1
    sum += num
    puts "In Block: #{sum}"
end
puts "Outer Block: #{sum}"

#=>     In Block: 11
#=>     In Block: 21
#=>     Outer Block: 0
